Pattern Blocks

Pattern blocks are a traditional educational "manipulative" that allows students to explore geometrical patterns. Why should you do this on a computer rather than with the nice, tactile, easily available wooden blocks? The main reason is that the program includes a group command, which makes it much easier to build large patterns. The group command lets you group a set of blocks into a configuration that becomes a new block that you can clone, rotate, and otherwise manipulate to make a larger structure. This makes it easy to build up large patterns fast, and also lets you come up with interesting patterns through exploration.

NOTE: Java-capable browser required. The Java applet will require you to grant it access privileges.

The user interface for Patblocks is a little idiosyncratic, so here's a suggestion for getting started:

You can always drag out new blocks from the blocks on the left edge of the screen. You probably want to avoid the pentagon and octagon, at least to start.

The buttons at the top of the window put PatBlocks in a mode -- Drag, Rotate, Group, Clone, or Delete. Drag is the default. In Drag mode, you can also use the right mouse button to rotate.

The Group button is a little tricky. To make a group --

  • arrange the blocks as you like
  • click on the group button
  • click on the blocks you want to include in the group (their border will turn white)
  • when done, click on group again to form the group
  • the group appears in the palette, and the originals are removed from the main screen

The Window button moves the pattern block program out of the web page and into its own window, so that it can be enlarged for bigger and better patterns.

One handy hidden feature: in Drag mode, dragging with the right mouse button will actually do a rotate operation.
